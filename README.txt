NOTE Java 11 is required for this application.

Known error: if there are 2 different versions of java on your path there is a possibility of an error thrown. All references
to Java 8 must be removed from system and environment variables in the class path.

How to run:

1.) Go to runway_redeclaration_jar folder
2.) double click the runway-redeclaration.jar (there are some extra dependencies in here, do not delete them)

Run through terminal

1.) cd runway_redeclaration_jar 
2.) java -jar runway-redeclaration.jar

/Application Demonstration - Contains 2 fully acted scenarios as well as additional video showing error handing and accessibility features + extras
